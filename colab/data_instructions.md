## External data files required to run the notebook

Two Excel files need to be uploaded manually when running the Colab notebook.
These files are not included in this repository for copyright reasons.
Download them from the links below before running the notebook.

File 1 — TCGA survival data (Liu et al. 2018)
Filename: 1-s2_0-S0092867418302290-mmc1.xlsx
Download from: https://ars.els-cdn.com/content/image/1-s2.0-S0092867418302290-mmc1.xlsx
Upload when: Section 2 of the notebook prompts you

File 2 — TME subtype labels (Bagaev et al. 2021)
Filename: 1-s2_0-S1535610821002221-mmc6.xlsx
Download from: https://ars.els-cdn.com/content/image/1-s2.0-S1535610821002221-mmc6.xlsx
Upload when: Section 3 of the notebook prompts you

The spatial feature CSV files (TCGA and CPTAC) download automatically 
inside the notebook from the authors' GitHub repository.
No manual download needed for those.
